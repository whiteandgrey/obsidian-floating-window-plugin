# Obsidian Floating Window Plugin

# Obsidian 悬浮窗口插件

---

## What is this plugin for?

## 这个插件是用来做什么的？

This plugin allows you to display selected text passages in independent floating windows while maintaining the exact same rendering and theme style as the original document.

这个插件允许您在独立的悬浮窗口中显示选中的文本段落，保持与原文档完全一致的渲染效果和主题风格。

Perfect for fixing important paragraphs on screen for reference while reading long documents.

非常适合在阅读长文档时，将重要段落固定在屏幕上进行参考对比。

You can extract key information from different documents into floating windows for easy comparison and cross-reference.

您可以从不同文档中提取关键信息到悬浮窗口中，方便对比和交叉参考。

The floating window supports complete Markdown rendering, including headings, code blocks, lists, tables, MathJax formulas, and more.

悬浮窗口支持完整的Markdown渲染，包括标题、代码块、列表、表格、MathJax公式等。

## How to use

## 如何使用

**Step 1: Select text**

**步骤一：选中文本**

Select any text passage in a Markdown document.

在Markdown文档中选中任意文本段落。

**Step 2: Open floating window**

**步骤二：打开悬浮窗口**

Use one of the following methods:

使用以下方法之一：

- Right-click on the selected text and choose "Open in floating window"
- 右键点击选中文本，选择"Open in floating window"

- Press `Alt+w` keyboard shortcut
- 按下 `Alt+w` 快捷键

**Step 3: Interact with the window**

**步骤三：操作窗口**

- **Move**: Click and drag anywhere on the window
- **移动**：点击窗口任意位置并拖动

- **Resize**: Drag window edges or corners
- **调整大小**：拖动窗口边缘或角落

- **Close**: Click the × button (top-right corner)
- **关闭**：点击右上角 × 按钮

- **Jump to source**: Click the → button (top-left corner) to jump back to the original paragraph
- **跳转原文**：点击左上角 → 按钮跳转回原文档对应段落

## Key Features

## 核心功能

- **Intelligent Sizing**: Automatically calculates optimal window size based on content, prioritizing full line display without scrollbars
- **智能尺寸调整**：根据文本内容自动计算最佳窗口大小，优先完整显示整行内容，避免滚动条

- **Smart Positioning**: Windows automatically position at the bottom-right of selected text, avoiding overlap with your selection
- **智能定位**：窗口自动定位到选中文本末行右下方，避免遮挡选中的内容

- **Auto Cleanup**: When you close a document, all related floating windows are automatically closed
- **自动清理**：关闭文档时，所有相关的悬浮窗口会自动关闭

- **Multi-language Support**: Supports Chinese and English interface
- **多语言支持**：支持中文和英文界面
